**# Dataset Information
**
Dataset: Online Retail Dataset

Source:
https://archive.ics.uci.edu/dataset/352/online+retail

Records: 525,936

Description:
This dataset contains transactions occurring between 01/12/2010 and 09/12/2011 for a UK-based online retail company.

Columns:
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

Note:
The original dataset is not included in this repository due to file size limitations.
